[Q] edit custom vm options for mac

Posted on 2020-10-08，1 min read

一、为什么会遇到这个问题？

编辑studio.vmoptions设置内存大小时不小心输入错误的字符，导致AS打不开了

二、解决了吗？

解决了，我的Android studio版本是4.1，配置文件路径和之前的不一样，让我好一顿找

Android studio4.1之前：~/Library/Preferences/AndroidStudio xxx
Android studio4.1及4.1之后：~/Library/Application Support/Google/AndroidStudio xxx
